# 🌿 English → Kannada Translator

A clean, minimal web app that translates English text to Kannada in real time — built with Python, Flask, and Google Translate via `deep-translator`.

![Python](https://img.shields.io/badge/Python-3.x-blue?style=flat-square)
![Flask](https://img.shields.io/badge/Flask-3.x-lightgrey?style=flat-square)
![deep-translator](https://img.shields.io/badge/deep--translator-Google-orange?style=flat-square)

---

## ✨ Features

- Instant English to Kannada translation
- Soft, warm UI with DM Serif Display + DM Sans fonts
- Click translated text to copy to clipboard
- Character counter (up to 1000 characters)
- Keyboard shortcut — `Ctrl + Enter` to translate
- Responsive layout — works on mobile too

## 🚀 Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Disha286/Translator.git
cd Translator
```

### 2. Install dependencies

```bash
pip install flask deep-translator
```

### 3. Run the app

```bash
python app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000
```
